CEF and WPF rock!

A SchemeHandler in Cef is a class that can handle requests going below each type of url. Consider the following urls:

  • local://web/index.html
  • custom://web/home.html
  • custom://web/index.html
  • KB://web/home.html

The list above shows 4 urls from 3 schemes. Each scheme: "resource", "custom", "KB" can be implemented inside a specific SchemeHandler class that implements the ISchemeHandler interface.

And all 3 classes could be created by one factory class that implements ISchemeHandlerFactory interface.
